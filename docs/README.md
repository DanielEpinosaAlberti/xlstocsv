# Generated Documentation

This folder contains the HTML documentation output aligned with the JSDoc annotations in the source code.

Open `docs/generated/index.html` to view the generated reference.

If Node.js is available in your environment, you can regenerate the site from the repository root with:

```bash
npm install
npm run docs
```

The root `package.json` and `jsdoc.json` files define the generation command and output directory.
